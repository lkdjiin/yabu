I’m parsing the ‘configuration/directories.conf’ which contains files and directories to add (or remove) to the backups. From me you can get two lists :

  1. files/directories to include to the backup

  2. files/directories to exclude from the backup

Examples:

: 

c = DirConfig.new 'configuration/directories.conf'
includes = c.files
excludes = c.filesToExclude
